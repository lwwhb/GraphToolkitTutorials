# GraphToolkit 教程系列

Unity 6 GraphToolkit 完整教程，从基础到高级应用。

## 教程概览

### 阶段一：基础入门

#### [教程1: Hello Graph - 简单计算器图形](01_HelloGraph.md)
**难度**: ⭐
**时间**: 1-2小时

学习GraphToolkit的核心概念：
- Graph、Node、Port基础
- 数据流评估
- ScriptedImporter使用
- 创建简单的数学运算图形

**关键收获**: 理解图形系统的基本架构

---

**关键收获**: 掌握数据流图形的设计模式

## 项目设置

### Unity版本
- **要求**: Unity 6000.0.5a8 或更高
- **原因**: GraphToolkit是Unity 6的内置模块

### 项目配置
1. 创建新的Unity 6项目
2. 选择URP模板（教程9-10需要）
3. 克隆或下载本教程仓库
4. 导入到项目中

### 验证安装
```csharp
// 在任意Editor脚本中测试
using Unity.GraphToolkit.Editor;

// 如果没有编译错误，说明GraphToolkit可用
```

## 教程约定

### 代码风格
- 使用C# 9.0特性
- 遵循Unity命名规范
- 详细的中文注释

### 文件组织
```
Assets/Tutorials/
├─ 01_HelloGraph/
│  ├─ Editor/           # 编辑器代码
│  ├─ Runtime/          # 运行时代码（如果需要）
│  └─ Examples/         # 示例资产
├─ 02_DataFlow/
└─ ...
```

### 命名规范
- **Graph类**: `[功能]Graph`（如CalculatorGraph）
- **Node类**: `[功能]Node`（如AddNode）
- **文件扩展名**: 简短且描述性（如.calc, .texgraph）

## 常见问题

### Q: 为什么使用Unity 6？
A: GraphToolkit是Unity 6的新特性，作为内置模块提供。旧版本无法使用。

### Q: 可以在生产项目中使用吗？
A: Unity 6目前处于Alpha/Beta阶段，建议等待正式版。但教程中的设计模式和概念可以应用到任何版本。

### Q: 教程代码的许可证？
A: MIT许可证，可自由使用和修改。

### Q: 如何获取帮助？
A:
1. 查看教程文档中的"常见问题"部分
2. 阅读代码注释
3. 参考Unity官方文档

### Q: 教程会更新吗？
A: 会随着Unity 6的正式发布和GraphToolkit的API变化进行更新。

## 学习建议

### 1. 动手实践
不要只看文档，一定要自己编写代码和创建图形。

不要死记硬背API，理解背后的设计模式和架构思想。

不要跳过基础教程，它们是后续学习的基础。
